# 🍔 DeliveryInsights: Análise NLP em Dados do iFood

Bem-vindo ao repositório oficial do nosso Projeto Aplicado II. Este projeto tem como objetivo aplicar técnicas de Ciência de Dados e Processamento de Linguagem Natural (NLP) para analisar o ecossistema de restaurantes da plataforma iFood.

## 🎯 Objetivo do Projeto

Desenvolver um modelo de Aprendizado de Máquina baseado em texto para analisar categorias, segmentos e nomes de restaurantes. O foco é identificar padrões textuais que ajudem a entender o perfil dos estabelecimentos e otimizar os sistemas de recomendação ou categorização em plataformas de delivery.

## 🗂️ Base de Dados

Os dados utilizados neste projeto são de domínio público e foram extraídos do Kaggle.
* **Dataset:** [iFood Restaurants Data](https://www.kaggle.com/datasets/ricardotachinardi/ifood-restaurants-data)
* **Formato:** `.csv`
* **Foco da Análise:** Variáveis textuais (nomes, categorias, descrições) e sua relação com as avaliações e performance.

## 🛠️ Tecnologias e Bibliotecas Utilizadas

* **Linguagem:** Python
* **Manipulação de Dados:** Pandas, NumPy
* **NLP:** NLTK, Spacy (ou bibliotecas similares a definir)
* **Machine Learning:** Scikit-learn
* **Visualização:** Matplotlib, Seaborn
